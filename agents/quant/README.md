# agents/quant

Quant-finance specialists. These agents lean into algorithmic trading, market microstructure, AFML methods, and finance-specific data quality. For the domain-neutral variants, see `agents/core/`.

| Agent | When to invoke |
|---|---|
| `quant-trading-architect.md` | Algorithmic trading systems, backtest engines, live trading infrastructure, strategy development, market microstructure, derivatives pricing, execution optimization. Sister to `core/systems-architect.md` (domain-neutral). |
| `quant-researcher.md` | Quantitative strategy research, alpha generation, AFML methods (triple-barrier, CUSUM, sample weights), feature engineering for financial data, IC/ICIR analysis. Sister to `core/research-engineer.md`. |
| `data-quality-scientist.md` | Rigorous data validation specifically for finance: OHLCV integrity, look-ahead detection in market data, IC measurement, stationarity for crypto/equity series. Sister to `core/data-quality-engineer.md`. |
| `ml-systems-architect.md` | High-performance ML for finance: SIDE/META models, sequential architectures (iTransformer, Mamba, TFT), regime-conditional ensembles, walk-forward validation. Sister to `core/ml-engineer.md`. |
